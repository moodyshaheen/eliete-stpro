# ⚡ حل سريع لمشكلة 404

## الخطوات السريعة:

### 1. اذهب إلى Settings → Pages
   - رابط مباشر: `https://github.com/moodyshaheen/eliete-stpro/settings/pages`
   - في **Source**، اختر: **GitHub Actions** ✅
   - احفظ

### 2. اذهب إلى Actions
   - رابط مباشر: `https://github.com/moodyshaheen/eliete-stpro/actions`
   - اضغط على **"Deploy to GitHub Pages"**
   - اضغط **"Run workflow"** → **"Run workflow"**
   - انتظر حتى ينتهي (2-5 دقائق)

### 3. بعد النجاح ✅
   - انتظر دقيقة واحدة
   - اذهب إلى: `https://moodyshaheen.github.io/eliete-stpro/`

---

## إذا لم يعمل:

### تحقق من:
- [ ] المشروع Public (وليس Private)
- [ ] GitHub Pages مفعل
- [ ] Source = "GitHub Actions" (وليس branch)
- [ ] Workflow نجح (علامة ✅)
